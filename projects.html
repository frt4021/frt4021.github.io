My Projects

Explore some of the projects I've worked on

IoT Security Companion Application Vulnerabilities and Simulations

This project focuses on identifying vulnerabilities in companion apps that directly affect IoT devices. The aim is to create a testing environment to simulate the behavior of IoT devices and analyze their firmware, detecting security flaws without the need for physical devices.

Parallelization of Edge Detection Algorithms

This project enhances the performance of the Robert edge detection algorithm by using CUDA for GPU, MPI for CPU cores, and OpenMP for multi-threading. It compares various parallelization techniques to assess acceleration and efficiency.

Spell Check Algorithm

A spelling check system developed based on the QWERTY keyboard layout using a graph and Levenshtein distance algorithm. The project focuses on detecting user errors and suggesting corrections, taking into account the Turkish QWERTY keyboard layout and character similarities.
